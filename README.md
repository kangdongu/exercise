# Habit

Habit은 운동습관을 만들어 주기위한 앱입니다. <br />
운동습관은 체력, 근력, 체지방같이 외적인 부분뿐만 아니라<br /> 
내가 하기로한 운동을 실천하면서 무기력함을 없애고<br /> 나에대한 신뢰, 성공습관, 자존감을 상승시켜주며 많은 사람들의 삶에 도움을 주기위한 앱입니다.
